# Grunt Project
Para inicializar proyectos con Gulp

## Módulos
```
grunt-browser-sync
grunt-contrib-jshint
grunt-contrib-sass
grunt-contrib-uglify
grunt-contrib-watch
grunt-notify
```
## Árbol de archivos
```
|- app/
  |- css/
  |- index.html
  |- js/ 
  |- scss/
|- dist/
|- Gruntfile.js
|- node_modules/
|- package.json
```

## 1.- Instalación
```
npm install
```
## 2.- Ejecutar la aplicación
```
grunt
```
